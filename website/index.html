Questions

FAQ.

Does it work offline?

Yes. thorn is a PWA — it works without a connection. GitHub sync and AI review require connectivity.

Is there a native desktop app?

In the works. Follow the GitHub repo to be notified when it ships.

Where are my files?

In your GitHub repo if you use sync, or in the browser's local storage if you don't. Nothing is stored on our servers.

Can I self-host it?

Yes. The entire app is a single HTML file. Open it in any browser, or serve it with any static host.

Is there an iOS or Windows version?

Not yet. The web app runs everywhere — open thorn.html in any browser.

What AI models work?

Any OpenAI-compatible API: GPT-4o, Claude (via OpenRouter), Llama 3 (via Groq or Ollama), Gemini (via OpenRouter), and more.
